# Spring Boot Web Blog Project

A modern web blog application built with Spring Boot that allows users to create, read, update and delete blog posts.

## Technologies Used

- Java 17
- Spring Boot 3.x
- Spring Security
- Spring Data JPA
- MySQL
- Maven
- Bootstrap 5

## Features

- User authentication and authorization
- Blog post management (CRUD operations)
- Comment system
- Responsive design
- Category management
- Search functionality
- User profile management

## Prerequisites

- JDK 17 or later
- Maven 3.6+
- MySQL 8.0+
